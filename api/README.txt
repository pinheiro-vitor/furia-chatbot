Esta pasta contém a API Express usada como backend local para simular os dados do chatbot, substituindo completamente o Supabase Functions. O endpoint principal é /api/hltd-data.

Para rodar:
1. Instale as dependências (express, cors): npm install express cors
2. Execute: node api/hltd-data.js

Adapte ou expanda conforme necessidade do seu projeto.
